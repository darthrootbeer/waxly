What is Waxly?

Waxly is infrastructure, not a product. It's a single source of truth for vinyl terminology, designed for:

  • AI & LLMs - Machine-readable, structured data
  • Developers - Free public API at /v1/
  • Researchers - Open dataset (CC BY-SA 4.0)
  • Humans - Fast, accessible terminology reference

API Access

Free, open, read-only API:

GET /v1/term/{slug}    # Get single term
GET /v1/terms          # List all terms
GET /v1/search?q=      # Search terms
GET /v1/random         # Random term
